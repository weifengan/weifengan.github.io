NGUI组件之输入文本(UIInput) 2016-10-15

输入框介绍

输入框是再Label的基础上扩展出来的一种组件，主要的用途是接收用户的输入文本,可以输入单行或者多行文本，常用于注册界面的数据校验。

输入框的创建

参考效果:

1)创建一个UISprite作为输入框背景，并命名为Input.

2)在Input下添加一个Label,用于显示输入的文字。

3)在Input游戏对象上添加BoxCollider和UIInput组件，并设置相应的属性。

输入框的常用属性

Label 用于显示输入文字的Label

StaringValue 程序运行时显示的文本内容

Saved As

ActiveTextColor 激活颜色

Inactive Color 未激活颜色

CaretColor 括号颜色

Selection Color 选中颜色

Input Type 文本输入类型

  • Standard 标准输入
  • Password 输入密码
  • Validation 校验类型
    • None 不校验
    • Integer 数字
    • Float 小数
    • Alphanumeric： 字母+数字
    • Username :不能使用特殊字符，不能使用汉字，可以字母和数字
    • Name 首字母自动大写,不能数字,不能特殊字符,不能中文

OnReturnKey 回车键操作

  • Submit 提交
  • New Line 开始新行

OnSubmit

当OnReturnKey属性设定为Submit时，用户按下回车时进行调用

OnChange

当用户进行字符输入时进行回调
