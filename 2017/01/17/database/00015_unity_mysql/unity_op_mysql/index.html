准备工作

如果想在Unity中连接MySQL数据库，需要大家提前做好准备

  • mysql.data.dll 数据库连接类（点击 此处下载)

  • 调整Unity工程的发布设置中Player Settings里的参数

    将API Compatibility Level调整为 .Net 2.0

  • 数据库SQL文件（点击 此处 下载SQL文件）
